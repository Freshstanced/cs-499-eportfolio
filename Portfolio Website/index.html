Professional Self-Assessment

As I complete the Computer Science program, my ePortfolio represents both my technical growth and my career direction. Throughout the program, I have developed skills in software design, full stack development, databases, algorithms, security, and technical communication.

For my CS 499 capstone, I selected my CS-465 Travlr Full Stack Application as my main artifact. Travlr is a travel website that includes a Node.js and Express backend, MongoDB database models using Mongoose, Handlebars views, API routes, authentication logic, and an Angular admin application.

I enhanced this artifact across three categories: software design and engineering, algorithms and data structures, and databases. These enhancements demonstrate my ability to evaluate existing code, improve application structure, process and validate data, strengthen database models, and explain technical decisions clearly.
